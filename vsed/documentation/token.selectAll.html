# token.selectAll()

Examples

 

// select all token with category 1
var mytokens = scene.selectAll("category",1);
// add texture baloon to all this tokens
mytokens.attr('fillStyle','red');

Description

Token.attr(), is a getter and setter for tokens attributes. If the first parameter alone, then attr(), return the attribute. If the first parameter and the second, then the token attribute named by the first parameter took the value of the second parameter. Token.attr() is chainable.

Syntax


token.attr('key') 		// return the key attribute
token.attr('key',value)	// set the key withe value, and return the token 
token.attr('key',value).attr('key2',value2) // possibility to chain

Parameters

Parameter in bold are needed.
key corresponde to the token attributes, or your custom one
value value of the attribute

Return

The token object, or the requested attribute.
